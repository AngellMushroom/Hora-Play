# Hora-Play
Hora Play é um site de notícias de jogos e tecnologia nesta área. Além da informação, seu objetivo é guiar e ensinar pessoas que não possuem contato nesta área.
Para o FrontEnd foi utilizado HTML e CSS, para o BackEnd somente JavaScript. As maiores dificuldades neste código foram conectar as paginas, implementar uma API, organizar os códigos em pastas (não conectam e dão erro).
